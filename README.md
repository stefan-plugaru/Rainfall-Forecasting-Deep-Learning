# Rainfall Forecasting with Deep Learning
MSc Financial Mathematics project exploring GRU, LSTM and Transformer models for multi-region UK rainfall forecasting.

## Main Ideas
- Preprocessing of historical rainfall time-series
- Models: GRU, LSTM, Transformer
- Evaluation metrics: MAE, RMSE, correlation coefficient

## Tech Stack
- Python, NumPy, pandas
- scikit-learn
- TensorFlow / PyTorch
